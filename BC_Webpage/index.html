What is Bitterness Classifier

Determining the presence of a bitter taste is important for many reasons. From pharmaceutical industries, to Food industries, as well as having therapeutic potential too. However, bitter taste is complex: the estimations are that there are thousands or more of bitter compounds, and it is known that bitter compounds are very diverse in terms of their chemical structure. To date, there are not many CS-based solutions for predicting bitterness. Bitterness Classifier aims to benefit from the Machine Learning field in order to implement a classifier that is able to predict the bitter/non-bitter taste of chemical molecules. To this end we used different learning algorithms, such as Random Forests, and Neural Networks, using ML libraries (such as Scikit-learn) and data from Prof Niv lab: BitterDB, a large public repository of bitter compounds as positive set, and in house non-bitter set as negative set.

BC is a tool capable of determining the presence of bitter taste in chemical molecule based on the chemical descriptors of it. BC uses the learning algorithms which are designed to work with the input descriptors in order to predict whether the input molecule(s) is bitter or not.

The input required by BC is an Excel file (.xls, or .xlsx) containing the SMILES alongside the 60 descriptors agreed upon of the molecules. Upon completed prediction, BC engine will return an Excel file for download containing the Index of each molecule alongside its prediction of taste (bitter, or non-bitter).
